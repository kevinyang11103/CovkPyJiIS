# 校园社团信息管理系统 java431

## 项目概述

本系统是基于 Spring Boot 开发的校园社团信息管理系统。采用前后端分离的技术架构，前端使用 Vue，后端使用 Spring Boot，实现了社团信息的便捷管理与高效互动。

## 技术栈

- **开发语言：** Java
- **前端技术：** Vue
- **后端技术：** Spring Boot
- **数据库：** MySQL 5 或 8

## 功能模块

### 后台管理

- **管理员：**
  - 登录
  - 个人中心
  - 学生管理
  - 社长管理
  - 社团分类管理
  - 社团信息管理
  - 加入社团管理
  - 社团成员管理
  - 社团活动管理
  - 活动报名管理
  - 系统管理

- **学生：**
  - 登录
  - 个人中心
  - 加入社团管理
  - 活动报名管理
  - 我的收藏管理

- **社长：**
  - 登录
  - 个人中心
  - 社团信息管理
  - 加入社团管理
  - 社团成员管理
  - 社团活动管理
